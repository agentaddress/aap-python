# Security Policy

## Reporting a Vulnerability

Please report suspected security vulnerabilities privately by emailing
security@agentaddress.org.

Do not open public GitHub issues for suspected vulnerabilities.

Include enough detail to reproduce the issue, affected versions, and any
proof-of-concept code if available. We will acknowledge reports within 7 days
and coordinate a fix and disclosure timeline where appropriate.

## Supported Versions

This project is pre-1.0. Security fixes are applied to the latest released
version.
